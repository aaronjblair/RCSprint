## Key decisions (+ why)
- Distribution is **PWA + Windows .exe only** (no native iOS/Mac/Android — Windows box, no Apple hardware/account; iOS can't sideload). Binaries ship as **GitHub Release assets**; `release/` is gitignored.
- Arcade scope locked by the user: **performance pickups only (no weapons)**, **top-3 + continues**, **light-tree start in both modes**.
- `scripts/build-win.mjs` builds electron to a temp dir then copies the installer to `./release` — works around an **EPERM** the project tree throws on electron-builder's populated-dir rename.
- **Never `git add -A`** here: a concurrent session leaves untracked strays (`tools/*`, `public/superjay-logo.png`) — stage specific files only.
