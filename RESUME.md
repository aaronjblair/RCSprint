## Key decisions
- **Only the cars + track are 1:10**; everyone/everything else is real-world size (1u ≈ 1ft). Unchanged.
- Sound is **player-car only**, on by default after the first gesture, with a mute toggle — not a
  full-field engine mix (kept subtle and cheap).
- Sound is **pure Web Audio** (no audio file shipped); degrades to a silent no-op if `AudioContext`
  is unavailable (headless).
